# Clipboard Tool

A simple, browser-based tool to read from clipboard, copy text to clipboard, cut (copy + clear), and clear text — no installation required!


## Features

- Paste text from clipboard with one click
- Copy text to clipboard
- Cut text (copies to clipboard and clears the field)
- Clear all text instantly
- Status messages after each action
- Clean, colorful, responsive buttons
- Single-file HTML + CSS + JavaScript (no frameworks)

## Requirements

Any modern browser:
- Chrome / Edge / Firefox / Safari / ...

**Important:** The first time you click "Paste from Clipboard", the browser will ask for permission → click **Allow**.

## How to Use

1. Download `clipboard.html` (or clone the whole repository)
2. Double-click the file to open it in your browser
3. Use the buttons:

   - **Paste from Clipboard** → reads whatever is currently in your clipboard
   - **Copy to Clipboard** → copies the text in the box
   - **Cut to Clipboard** → copies the text and then clears the box
   - **Clear Text** → removes everything from the text area
